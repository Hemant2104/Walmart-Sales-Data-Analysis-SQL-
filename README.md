# Walmart-Sales-Data-Analysis-SQL-

Information about Project

This repository contains a project focusing on Walmart sales data. The project utilizes MySQL to explore and analyze various aspects of sales trends and popular product lines within the dataset. SQL queries are employed to answer specific questions about sales patterns and to calculate important metrics such as total sales and average sales.
The primary objective of this project is to analyze sales data from Walmart using MySQL queries to gain insights into sales trends and popular product lines
Uses aggregation functions such as SUM, COUNT, and AVG are utilized to calculate important metrics including total sales, average sales, and more.
Also use GROUP BY and ORDER BY clauses are used effectively to organize and sort the data for meaningful analysis and visualization.
